# Тестовое задание SolarLab (Уровень 4)

Серверная часть задания. Клиентская часть: https://github.com/P0lapa/try-React

### Требования
- Docker + Docker Compose
- Java 21
- (для фронтенда) Node.js 24 и npm 11

***

Далее пойдёт небольшая инструкция по развёртыванию:

***

1. Скопировать данный репозиторий на локальную машину
2. В папке DB находится docker-compose файл. Пропишите команду "docker-compose up", открыв данную папку в терминале.
3. Далее запустите Spring сервер( после данного этапа уже можно открыть и тестирова API по ссылке: http://localhost:8080/swagger-ui/index.html )
4. Скопируйте на локальную машину клиентскую часть (требует наличия node 24+ и npm 11+)
5. Пропишите в корневой папке проекта(клиентской части) команду "npm run dev"

Поздравляю проект развёрнут!
